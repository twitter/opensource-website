Data Infrastructure for Experiment Support

Explore Datasets

The responsible and innovative use of data is instrumental in today's digitalised media industry. The EU-funded MediaFutures project is a virtual European data innovation hub to support entrepreneurial and innovative projects to reshape the media value chain. By delivering data and experimentation facilities to the winners, the project will showcase and improve their ideas.
